## 🎯 Overview
EduBuddy combines intelligent tutoring, resource management, community collaboration, and gamification into a seamless educational experience. Whether you're studying programming, databases, math, or AI, EduBuddy provides the tools to learn efficiently, stay consistent, and collaborate effectively.

---

## ✨ Key Features
| Category | Highlights |
|----------|------------|
| 📚 **Smart Resource Management** | Organize courses/TDs/TPs by sector & year, smart search, download tracking, community sharing |
| 🤖 **AI-Powered Assistant** | 24/7 academic support, code assistance, complex topic simplification, multi-subject coverage |
| 📎 **File Upload & Analysis** | Upload PDF, DOCX, TXT, MD, CSV (<50MB). AI extracts text and answers document-specific questions |
| 🧠 **Context-Aware Memory** | AI remembers which file you're discussing across messages, switches context intelligently, reduces re-uploads |
| 📊 **Activity Calendar** | GitHub-style streak visualization, color-coded intensity, current streak tracking, motivational UI |
| 🎨 **Modern UI/UX** | Dark theme, responsive design, GSAP animations, Radix UI components, accessibility-first |

- Python 3.10+
- PostgreSQL/SQLite (configured in backend)
